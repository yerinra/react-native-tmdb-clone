# TMDB
[The Movie Database](https://www.themoviedb.org) 클론 코딩 과제
<br><br/>
## 설치 및 실행
```
yarn install
npx expo start -c
```

## 사용한 라이브러리

- React Native
- TypeScript
- @reduxjs/toolkit
- axios
- expo
- expo-router
- nativewind
- appwrite
- @expo/vector-icons
<br><br/>
